# 🏠 Smart Rent Payment Dashboard

A modern and responsive **React-based web application** that simplifies rent payments with an intuitive dashboard, real-time calculations, and a clean user experience.

---

## ✨ Features

* 💡 **Dynamic Rent Calculation**
  Automatically calculates total rent based on selected months

* 📅 **Multi-Month Selection**
  Pay for Previous, Current, and Next months

* 👤 **Tenant Details Form**
  Capture tenant name, phone number, and rent details

* 💳 **Payment Method UI**
  UPI, Card, and Net Banking options (UI simulation)

* 🎉 **Payment Success Animation**
  Smooth success popup using animations

* 🔄 **Auto Reset After Payment**
  Form resets after successful payment for better UX

* 📱 **Fully Responsive UI**
  Works seamlessly across mobile, tablet, and desktop

---

## 🛠 Tech Stack

* ⚛️ React.js (Vite)
* 🎨 Tailwind CSS
* 🎬 Framer Motion
* 🧠 JavaScript (ES6)
---

## 🚀 Getting Started

### 1. Clone the repository

git clone https://github.com/Bhavyareddy2001/rent-payment-app.git

### 2. Navigate to project

cd rent-payment-app

### 3. Install dependencies

npm install

### 4. Run the app

npm run dev
